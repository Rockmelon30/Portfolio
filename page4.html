Kaggle Ames Housing Project Competition

Predicting real estate value of properties in AMES

July 21st, 2024 Corey Duncan

Brief Summary

This project was part of a Kaggle competition during my Data Science bootcamp at General Assembly, focused on building and optimizing regression models to predict home values using the Ames, California housing dataset.

The dataset contained a mix of numerical and categorical features representing historical sales data. Significant exploration and cleaning were required to address data quality issues before model development could begin.

I utilized methods such as linear regression, regularization techniques, and ANOVA tests to identify the most influential features in predicting sale price. Key predictors included Overall House Quality, Living Area, and Garage Area. Leveraging this analysis, I built a regression model that achieved 86% accuracy, placing 4th in my cohort.
